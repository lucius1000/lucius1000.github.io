linux之vim编辑器

发表于 2025-05-02 更新于 2026-05-13 分类于 computer 阅读次数：

1.vi和vim区别

vi是Unix和linux系统自带，经典文本编辑器，功能基础。

vim是vi的增强版，语法高亮、代码折叠、自动缩进等功能。

2.三种工作模式

执行命令，进入vim模式

2.1.一般命令模式

1. 光标移动

命令 作用
h 光标左移一个字符
j 光标下移一行
k 光标上移一行
l 光标右移一个字符
0 跳到本行行首
$ 跳到本行行尾
gg 跳到文档第一行
G 跳到文档最后一行
nG 跳到第 n 行，如 10G 跳第 10 行
Ctrl + f 向下翻一页
Ctrl + b 向上翻一页
Ctrl + d 向下翻半页
Ctrl + u 向上翻半页

2. 删除操作

命令 作用
x 删除光标所在后一个字符
X 删除光标所在前一个字符
dd 删除光标所在整行
ndd 删除光标开始往下 n 行，如 5dd
d$ 删除光标到行尾
d0 删除光标到行首

3. 复制与粘贴

命令 作用
yy 复制光标所在整行
nyy 复制往下 n 行，如 3yy
p 粘贴到光标下一行
P 粘贴到光标上一行

4. 撤销与重做

命令 作用
u 撤销上一步操作
Ctrl + r 重做（恢复撤销）

5. 区块选择（可视化模式）

命令 作用
v 字符区块选择，按光标范围选字符
V 整行区块选择，按行选中
Ctrl + v 矩形区块选择（列选择）
选中后 y 复制选中区块
选中后 d 删除选中区块

2.2.编辑模式

命令 进入编辑位置
i 光标前面插入
I 本行行首插入
a 光标后面插入
A 本行行尾插入
o 光标下一行新开空行插入
O 光标上一行新开空行插入
r 只替换一个字符，自动回到命令模式
R 持续替换模式，按 ESC 退出

2.3.底线命令模式（: 开头）

1. 保存与退出

命令 作用
:q 正常退出（无修改可退）
:q! 强制退出，放弃所有修改
:w 保存文件
:w! 强制保存（只读文件）
:wq 保存并退出
:x 保存并退出，同 :wq
:w 文件名 另存为新文件

2. 行号与设置

命令 作用
:set nu 显示行号
:set nonu 取消行号
:set hlsearch 搜索高亮
:set nohlsearch 取消搜索高亮

3. 查找与替换

命令 作用
/ 关键词 向下搜索关键词，n 下一个，N 上一个
? 关键词 向上搜索关键词
:s/old/new 当前行第一个 old 替换为 new
:s/old/new/g 当前行所有old 替换为 new
:%s/old/new/g 全文所有 old 替换为 new

4. 多文件 / 分屏

命令 作用
:n 切换到下一个文件
:N 切换到上一个文件
:files 查看所有打开的文件
:sp 水平分屏（上下）
:vsp 垂直分屏（左右）

5. 跳转行

命令 作用
:n 直接跳到第 n 行，如 :20
