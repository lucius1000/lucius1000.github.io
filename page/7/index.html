发表于 2025-05-10 更新于 2026-05-13 分类于 computer

1.什么是ShellScript

Shell 脚本：把一堆 Linux 命令、变量、判断、循环按顺序写到文本文件里，批量自动执行。

作用：

1.重复工作自动化

2.开机自启、定时任务

3.运维批量处理、日志分析、备份

执行脚本就是逐条执行里面的命令。

2.1. 脚本开头第一行必写

2.2. 注释

# 开头都是注释，程序不执行。

2.3. 简单模板 hello.sh

3.脚本三种执行方式

方式 1：给权限后直接运行（最常用）

方式 2：直接用 bash 解释器执行

方式 3：source 执行

特点：在当前终端进程执行，脚本里变量会影响当前终端。

4.脚本中的变量

1. 自定义变量

规则：等号两边不能有空格

2. 位置参数变量（重点）

执行脚本时后面跟的参数，自动接收：

变量 含义
$0 脚本本身名字
$1 $2 ... 第 1、第 2、第 3 个参数
$# 参数总个数
$@ 所有参数整体
$* 所有参数整体

示例: param.sh
